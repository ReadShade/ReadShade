
Generic Depth
=============

Introduction to Generic Depth
-----------------------------

The Generic Depth add-on makes depth effects in your games look better. To start, you need the add-on version of ReShade. Learn more about ReShade versions `here <../../reshade/reshadeversions>`_.

Downloading the Generic Depth Add-on
------------------------------------

To use the Generic Depth add-on, download it from the `official Discord channel <https://discord.com/channels/305472403977404416/1081018051526406195>`_. Make sure to pick the right version for your game so it works correctly.

.. admonition:: Note
   :class: tip

   Using the correct version of the add-on is very important for it to work well and avoid problems.

Once you have the right version, you can install the add-on.
