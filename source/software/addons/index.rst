
Addons
======

This section provides guides and information about various add-ons that enhance the functionality of ReShade, particularly focusing on tools that allow for greater control over in-game visual effects and depth perception.

Key topics include:

- **Generic Depth**: Learn how to use the Generic Depth add-on to enhance depth effects in your games. This guide covers downloading the correct version and installation, ensuring better visual immersion.
- **Reshade Effect Shader Toggler (R.E.S.T)**: A powerful tool for isolating game effects and applying ReShade shaders to specific parts of a game. This guide covers installation, configuration, enabling 3D shaders like SuperDepth3D, isolating UI elements, and sharing configurations.
- **Shader Toggler**: A detailed guide on how to install and use the Shader Toggler add-on with ReShade. This powerful tool allows users to disable or toggle specific shaders within games, such as chromatic aberration, to improve visual clarity or performance. The guide covers installation, configuration, setting up shortcuts, and saving custom shader toggle groups.
