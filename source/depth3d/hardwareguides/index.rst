Hardware Guides
===============

This section provides hardware guides for using SuperDepth3D with specific display technologies. It includes a guide for "Simulated Reality (SR) Displays," detailing both easy and manual installation methods for ReShade and 3DGameBridge, along with important notes on monitor and resolution settings. Additionally, it offers a guide for "VITURE AR Glasses," providing setup instructions for both Intel and NVIDIA GPUs, covering driver updates, software installations, and display configuration steps to enable stereo 3D gaming.
