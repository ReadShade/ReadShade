Generic Depth¶

Introduction to Generic Depth¶

The Generic Depth add-on makes depth effects in your games look better. To start, you need the add-on version of ReShade. Learn more about ReShade versions here.

Downloading the Generic Depth Add-on¶

To use the Generic Depth add-on, download it from the official Discord channel. Make sure to pick the right version for your game so it works correctly.

Note

Using the correct version of the add-on is very important for it to work well and avoid problems.

Once you have the right version, you can install the add-on.
